### Featured Projects

**[CareerFit Radar: Full-Stack RAG Pipeline for Job Discovery](https://github.com/farazkhan-05/Careerfit-radar)**

A job discovery tool that reads a resume and finds roles that fit. It parses a PDF, DOCX, or text resume, builds a candidate profile with Gemini, and pulls current job listings from the web through Tavily across approved sources. Each role gets a seven-signal fit score and a short gap analysis, so every match comes with a reason and a sense of what to improve. Saved roles can be tracked through application stages and exported to CSV.

The frontend is React 18 and Vite with Tailwind and TanStack Query. The backend uses FastAPI with Neon Postgres and pgvector for embeddings, plus SQLAlchemy and Alembic for data and migrations. A LangGraph workflow handles importing, deduplicating, embedding, and ranking the roles. It runs as separate Dockerized services on Google Cloud Run, with an Nginx proxy that keeps the backend token server-side.

**[SpanishAmigo: Production RAG Application with Real-Time AI Chat](https://github.com/farazkhan-05/LangGraph-Orchestrated-AI-Workflow-with-SSE-Streaming-Secure-Persistence)**

A Spanish learning web app with structured lessons and an in-app AI tutor. Lessons run through a slide-based player covering context, translation reveal, and practice quizzes. The tutor answers questions alongside the lesson flow with streamed responses, voice input, and speech playback.

The frontend is React 19 and Vite with Material UI and Tailwind. The backend uses FastAPI with Postgres, Firebase Authentication, and Alembic for migrations. The tutor is built on LangGraph with Google Gemini, with optional lesson retrieval using pgvector embeddings. Guests use the app with local progress; signed-in users sync to the backend.
